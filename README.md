meteor-hamlet-handlebars
========================

meteorite package for @gregwebs hamlet.js

hamlet is a jade inspired, HTML friendly syntax. 

so, instead of 

```
p.foo#class
  | text
  | text
h1
  | ooo
  em foo
```

in hamlet you'd write:

<p.foo#class>
  text
  text
<h1>
  ooo
  <em> foo
```
